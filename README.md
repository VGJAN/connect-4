# 🎮 Connect 4 Game

A responsive browser-based Connect 4 game built using **HTML, CSS, and vanilla JavaScript**.  
This project focuses on structured game logic, interactive UI design, and smooth animations.

🔗 **Live Demo:**  
https://vgjan.github.io/connect-4/

---

## ✨ Features

- Two-player gameplay (Red vs Yellow)
- Win detection (horizontal, vertical, diagonal)
- Draw detection
- Score tracking across rounds
- Alternating starting player
- Animated disc drop with bounce effect
- Interactive cursor preview
- Visual highlight for winning discs
- Responsive design using CSS variables
- Smooth UI interactions and feedback

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Variables, Flexbox, Animations)  
- JavaScript (ES6+)  
- Font Awesome (icons)

---

## 🎮 How to Play

1. Players take turns dropping discs into a column  
2. Click on a column to place your disc  
3. The disc falls to the lowest available space  
4. First player to connect 4 discs wins  
5. Scores update automatically  
6. Starting player alternates each round  
7. Use the **Reset** button to start a new round
